# レビューガイド

ドキュメント種別: ガイド。権威: `docs/contracts/verification-lanes.md`。

## 共通レビュー質問

- 所有者コントラクトは何ですか?
- これは SoT の変更ですか、それとも投影の変更ですか?
- トポロジーは維持されていますか?
- 変更は 統一ノートサーフェス を維持していますか?
- 変更はキー入力ごとの AI 構造化を避けていますか?
- 変更はユーザーが書いたテキストを保護していますか?
- AI 操作はソースに裏付けられていますか?
- テストは検証レーンと整合していますか?
- 関連する非機能要件を満たしていますか?
- MVP 受け入れ条件の gap を作る、または閉じる変更ですか?

## 却下条件

- 永続的な AI チャットを MVP コアとして追加している。
- MVP に AI モード切り替えを導入している。
- Markdown を内部 SoT として扱っている。
- AI によってユーザーブロックを直接書き換えている。
- ソース/provenance のない 隠れた memory を追加している。
- `docs/contracts/mvp-acceptance.md` の未達項目があるのに MVP complete として扱っている。
