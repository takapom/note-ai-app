# エージェントワークスペース復旧ランブック

ドキュメント種別: ランブック。

## 症状

- Codex がタスクスコープ外のファイルを変更した。
- Superset ワークスペースに競合する編集がある。
- エージェントが検証前に停止した。

## 復旧

1. エージェントを一時停止または停止します。
2. 差分サマリーを取得します。
3. 変更されたファイルを所有者コントラクトと比較します。
4. スコープ外のファイルを元に戻します。
5. Codex に振り返りの作成を依頼します。
6. 同じ誤りが再発する可能性が高い場合にのみ、タスクプロンプトまたは AGENTS.md を更新します。
7. ブランチが汚染されている場合は、新しい Superset ワークスペースで再起動します。

## エスカレーション条件

- タスク承認なしにコントラクトファイルが変更された。
- 生成された投影がコントラクトと一致しない。
- テストなしに AI 操作スキーマが拡張された。
