# ADR-0003: Block / Section ドキュメントモデル

ドキュメント種別: 記録。

## 決定

アプリは内部 信頼できる唯一の情報源 として、アプリ固有の Block / Section モデルを使用します。Markdown はインポート/エクスポート専用です。

## 理由

AI 操作には安定した対象、ソーススパン、provenance、セクション単位の構造化が必要です。Markdown は壊れやすいテキストオフセットなしでは、正準な操作対象として信頼性高く機能できません。

## 結果

- H1/H2/H3 は構造的なセクション境界です。
- 大きなテキストと太字はスタイルであり、構造ではありません。
- AI Assist Blocks は origin `ai` を持つブロックです。
