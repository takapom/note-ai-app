# ADR-0002: AI 構造化トリガー

ドキュメント種別: 記録。

## 決定

MVP の AI 構造化は主にノートを閉じた後 / タブ切り替え後 / アプリ離脱後、および手動整理で実行されます。キー入力ごとには実行されません。

## 理由

即時の構造化は執筆フローを損ない、不安定な思考を早すぎる段階で構造化し、推論品質を下げ、トークンコストを増やします。

## 結果

- BlockChanged は保存と dirty マークのみを行います。
- 構造化ジョブはノート離脱 / 次回オープン時の復旧 / 手動整理から作成されます。
- 次回オープンダイジェスト が主要な表示パターンになります。
