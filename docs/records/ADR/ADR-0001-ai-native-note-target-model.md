# ADR-0001: AI ネイティブノートのターゲットモデル

ドキュメント種別: 記録。

## 決定

このリポジトリは、ユーザーが 1 つの統一ノートサーフェスとやり取りし、AI 由来の意味構造、メモリ、関係、操作は内部で分離される AI ネイティブノートアプリを対象にします。

## 理由

従来のノートアプリは、記憶、レビュー、手動構造化、コンテキスト接続、次のアクションの判断といった多すぎる責任を人間に負わせています。このアプリは、人間の判断と著作性を保ちながら、記憶と構造化の負担の一部を取り除きます。

## 結果

- UI は人間レイヤーと AI レイヤーを別々の主要サーフェスに分割してはいけません。
- AI は静かで編集可能でなければなりません。
- ユーザーが書いたコンテンツは 信頼できる唯一の情報源のままです。
